# Productivity Hub

Сайт-продуктивность из 6 независимых разделов ("миров"): Фокус, Задачи и цели,
Мысли, Привычки, Фитнес, Финансовый учёт. React + Vite, без бэкенда — все
данные хранятся локально в браузере.

## Стек

- React + Vite (JavaScript)
- react-router-dom — навигация между разделами
- zustand (+ persist) — настройки и состояние, сохраняются в localStorage
- react-i18next — интерфейс на русском, английском и узбекском
- jspdf / docx / xlsx — экспорт данных каждого раздела в PDF, Word и Excel

## Запуск локально

```
npm install
npm run dev
```

## Структура

```
src/
  layout/       — Sidebar, TopBar, переиспользуемые контролы
  pages/        — страницы разделов + Настройки
  store/        — zustand-стор настроек (язык, тема, видимые разделы)
  i18n/         — переводы (ru/en/uz)
  export/       — экспорт данных раздела в PDF/Word/Excel
  constants/    — конфиг разделов (sections.js)
  icons/        — набор SVG-иконок
```

## Разделы

Видимость каждого раздела можно включать/выключать в Настройках — скрытый
раздел пропадает из меню, но данные не удаляются. Там же — язык, тема и
экспорт данных отдельно по каждому разделу (PDF/Word/Excel).

Функционал самих разделов (Фокус, Задачи и цели, Мысли, Привычки, Фитнес,
Финансовый учёт) — в разработке, по одному, в любом порядке.
